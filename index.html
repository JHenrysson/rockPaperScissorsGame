Rock Paper Scissors game

When the program starts, a menu should appear with 2 options:
1. Create a new game
2. High score (extra points)
3. Exit

When a game is completed, the menu should appear again.

1.1. A new game will ask for a name and a move (rock, paper or scissors).
1.2. The computer randomly choose a move.
1.3. The result should be presented

2.1. A list with wins against the computer per user. Ties does not count.

3.1 The program should exit.
